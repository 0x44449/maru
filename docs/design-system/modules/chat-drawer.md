# Chat Drawer Design System

Atena Design System 기반의 채팅방 서랍(사이드 패널) 컴포넌트 규격.
채팅방 헤더의 햄버거 메뉴(≡) 탭 시 우측에서 슬라이드되어 나타난다.

> 기반: [`../DESIGN_SYSTEM.md`](../DESIGN_SYSTEM.md) / [`../tokens.json`](../tokens.json)
> 프리뷰: [`chat-drawer-preview.html`](./chat-drawer-preview.html)
>
> **상위 규격:** 아바타, Group Avatar는 `DESIGN_SYSTEM.md` 규격을 따른다.

---

# Part 1. 컴포넌트

---

## 1. Drawer Container (서랍 컨테이너)

화면 우측에서 슬라이드되는 패널과 백드롭으로 구성.

```
┌──────────┬───────────────────────┐
│          │                       │
│ Backdrop │       Drawer          │
│ (탭→닫기) │  ┌─────────────────┐  │
│          │  │    Header        │  │
│          │  ├─────────────────┤  │
│          │  │                 │  │
│          │  │    Content      │  │
│          │  │   (전체 스크롤)   │  │
│          │  │                 │  │
│          │  └─────────────────┘  │
└──────────┴───────────────────────┘
```

### 1-1. 패널

| 속성 | 값 |
|------|-----|
| 너비 | 300px |
| 높이 | 100dvh |
| 배경 | `surface` |
| 위치 | 우측 고정 |
| z-index | `modal` (1300) |
| 슬라이드 방향 | 우측에서 좌측으로 |
| 애니메이션 | `transition.slow` (300ms ease) |
| 구조 | flex column, header(`flex-shrink:0`) + content(`flex:1`, `overflow-y:auto`) |

### 1-2. 백드롭

| 속성 | 값 |
|------|-----|
| 배경 | `rgba(0, 0, 0, 0.5)` |
| z-index | `modal` - 1 (1299) |
| 동작 | 탭 시 서랍 닫기 |
| 애니메이션 | fade in/out `transition.normal` (200ms) |

## 2. Drawer Header (서랍 헤더)

서랍 상단의 고정 헤더. 닫기 버튼과 액션 버튼을 포함한다.

```
┌───────────────────────────────┐
│  [✕]                 [🔔] [⚙] │
│  44px                 44   44  │
└───────────────────────────────┘
           높이: 56px
```

### 2-1. 기본 규격

| 속성 | 값 |
|------|-----|
| 높이 | 56px |
| 배경 | `background` |
| 하단 보더 | `border` 1px |
| 좌측 패딩 | 4px |
| 우측 패딩 | 4px |
| 정렬 | 세로 중앙, 좌우 space-between |

### 2-2. 내부 요소

| 요소 | 규격 |
|------|------|
| 닫기 버튼 (좌측) | 44×44px 터치 영역, `x` 아이콘 24px |
| 알림 토글 버튼 | 44×44px 터치 영역, 아이콘 24px |
| 설정 버튼 | 44×44px 터치 영역, `settings` 아이콘 24px |
| 아이콘 색상 | `textSecondary` |

**알림 토글 아이콘:**

| 상태 | 아이콘 |
|------|--------|
| 알림 켜짐 (기본) | `bell` |
| 알림 꺼짐 | `bell-off` |

- 좌측: 닫기 버튼
- 우측: 알림 토글 + 설정 버튼 (나란히 배치)

## 3. Room Profile (대화방 프로필)

스크롤 영역 최상단에 표시되는 대화방 정보. 아바타와 이름을 중앙 정렬로 표시한다.

```
┌───────────────────────────────┐
│                               │
│         [Avatar][Avatar]      │
│           겹침 배치             │
│                               │
│         채팅방 이름             │
│                               │
└───────────────────────────────┘
```

### 3-1. 기본 규격

| 속성 | 값 |
|------|-----|
| 배경 | `background` |
| 패딩 | 32px 16px 24px |
| 정렬 | 수평 중앙 |

### 3-2. 아바타

| 속성 | 값 |
|------|-----|
| 크기 | 56px (`avatar.xl`) |
| Border Radius | 20px (둥근 사각형) |
| 보더 | 2px, `background` 색상 |

**1:1 채팅:** 상대방 아바타 1개.

**그룹 채팅:** 참여자 아바타 최대 2개 겹침 표시. `DESIGN_SYSTEM.md` Group Avatar의 크기 변형 적용.

| 속성 | 값 |
|------|-----|
| 겹침량 | 16px (두 번째 아바타가 첫 번째 위로) |
| 두 번째 아바타 z-index | 1 (위에 표시) |

### 3-3. 대화방 이름

| 속성 | 값 |
|------|-----|
| 텍스트 | 17px, semibold (600), `textPrimary` |
| 아바타-이름 간격 | 16px |
| 정렬 | 중앙 |
| 최대 줄 수 | 2줄, 말줄임 |

## 4. Media Gallery (미디어 갤러리)

채팅방에서 공유된 사진/동영상을 가로 스크롤로 미리보기.

```
┌───────────────────────────────────┐
│  🖼 사진/동영상                     │
│                                   │
│  [■] [■] [■] [■] [■] →  (가로스크롤) │
│                                   │
└───────────────────────────────────┘
```

### 4-1. 섹션 헤더

| 속성 | 값 |
|------|-----|
| 높이 | 44px |
| 좌우 패딩 | 16px |
| 아이콘 | `image` 20px, `textTertiary` |
| 아이콘-라벨 간격 | 8px |
| 라벨 | 15px, regular (400), `textPrimary` |
| 정렬 | 세로 중앙 |
| 탭 동작 | 전체 미디어 목록 화면 이동 |

### 4-2. 가로 스크롤 그리드

| 속성 | 값 |
|------|-----|
| 레이아웃 | 가로 스크롤 (`overflow-x: auto`) |
| 좌우 패딩 | 16px |
| 상단 패딩 | 0 (섹션 헤더 바로 아래 시작) |
| 하단 패딩 | 16px |
| 썸네일 크기 | 80×80px (고정) |
| 썸네일 간격 | 4px |
| 썸네일 Border Radius | 8px |
| 배경 (로딩/빈) | `surface` |
| 스크롤바 | 숨김 (`-webkit-scrollbar: none`) |

### 4-3. 동영상 표시

| 속성 | 값 |
|------|-----|
| 오버레이 | 반투명 검정 (`rgba(0,0,0,0.3)`) |
| 재생 아이콘 | `play` 24px, `#ffffff`, 중앙 배치 |
| 재생시간 | 11px, `#ffffff`, 우하단 4px 패딩 |

### 4-4. 빈 상태

미디어가 없을 경우 섹션 헤더만 표시하고 썸네일 영역은 숨긴다.

## 5. Navigation Item (네비게이션 아이템)

상세 화면으로 이동하는 메뉴 버튼. 파일, 링크 등의 항목에 사용.

```
┌───────────────────────────────┐
│  📁 파일                       │
│  🔗 링크                       │
└───────────────────────────────┘
```

### 5-1. 기본 규격

| 속성 | 값 |
|------|-----|
| 높이 | 44px |
| 좌우 패딩 | 16px |
| 아이콘 | 20px, `textTertiary` |
| 아이콘-텍스트 간격 | 8px |
| 텍스트 | 15px, regular (400), `textPrimary` |
| 정렬 | 세로 중앙 |
| 탭 피드백 | `surface` 배경 |
| 탭 동작 | 각 상세 목록 화면으로 이동 |

### 5-2. 항목

| 아이콘 | 라벨 | 동작 |
|--------|------|------|
| `file` | 파일 | 파일 목록 화면 이동 |
| `link` | 링크 | 링크 목록 화면 이동 |

네비게이션 아이템 사이에 구분선은 없다. 카드 내에서 미디어 갤러리와의 구분선만 존재.

## 10. List Items (리스트 아이템)

각 탭에서 렌더링하는 콘텐츠 아이템. 탭 종류에 따라 3가지 배리언트.

### 10-1. Media Grid (사진/동영상)

사진과 동영상을 3열 그리드로 표시.

```
┌────┬────┬────┐
│    │    │    │
│ ■  │ ■  │ ▶■ │
│    │    │    │
├────┼────┼────┤
│    │    │    │
│ ■  │ ■  │ ■  │
│    │    │    │
└────┴────┴────┘
```

| 속성 | 값 |
|------|-----|
| 컨테이너 패딩 | 2px |
| 열 수 | 3 |
| 아이템 간격 | 2px |
| 아이템 비율 | 1:1 (정사각형) |
| Border Radius | 0 (간격으로 구분) |

**동영상 오버레이:**

| 속성 | 값 |
|------|-----|
| 재생시간 | 11px, `#ffffff`, 우하단 6px 패딩 |
| 배경 | 하단 그라디언트 (`transparent` → `rgba(0,0,0,0.3)`) |

### 10-2. File Item (파일)

파일 목록의 개별 아이템.

```
┌──────────────────────────────────────┐
│  [📄]  프로젝트 기획서.pdf              │
│  40px  2.4 MB · 2024.03.15          │
└──────────────────────────────────────┘
```

| 속성 | 값 |
|------|-----|
| 높이 | 64px |
| 좌우 패딩 | 16px |
| 아이콘 영역 | 40×40px, `surface` 배경, 8px radius |
| 아이콘 | `file-text` 20px, `textTertiary` |
| 아이콘-텍스트 간격 | 12px |
| 파일명 | 15px, regular (400), `textPrimary`, 1줄 말줄임 |
| 메타 정보 | 13px, regular (400), `textTertiary` |
| 파일명-메타 간격 | 2px |
| 탭 피드백 | `surface` 배경 |

**메타 정보 형식:** `파일 크기 · 날짜` (예: "2.4 MB · 2024.03.15")

**아이템 간 구분선:**

| 속성 | 값 |
|------|-----|
| 높이 | 1px |
| 배경 | `border` |
| 좌측 들여쓰기 | 68px (아이콘 영역 우측 기준) |

링크를 OG 카드 형태의 2열 그리드로 표시. 메신저의 링크 프리뷰 카드(`messenger.md` #5-8)와 동일한 시각적 구조를 사용한다.

## 전체 레이아웃

카드형 섹션 그룹으로 구성. 각 카드는 `background` 배경에 둥근 모서리를 가지며, 카드 사이는 `surface` 배경으로 구분.

```
┌─ Drawer (surface 배경) ──────────┐
│  [2. Drawer Header] ─── 고정     │
├──────────────────────────────────┤
```

## 상세 화면 레이아웃

서랍의 미디어/파일/링크 탭 시 진입하는 전체 화면. 헤더 + 탭 + 콘텐츠로 구성.

```
┌──────────────────────────────┐
│  [8. Detail Header]          │
├──────────────────────────────┤
│  [9. Detail Tabs]            │
├──────────────────────────────┤
│                              │
│  [10. List Items]            │
│  (탭에 따라 다른 리스트)       │
│                              │
│         (스크롤 영역)         │
└──────────────────────────────┘
```

### 콘텐츠 영역

| 속성 | 값 |
|------|-----|
| 배경 | `background` |
| 스크롤 | `overflow-y: auto` |
| 빈 상태 | 콘텐츠가 없을 때 중앙에 안내 텍스트 표시 |

### 빈 상태

| 속성 | 값 |
|------|-----|
| 아이콘 | 각 탭 아이콘 40px, `textTertiary` |
| 텍스트 | 14px, regular (400), `textTertiary` |
| 아이콘-텍스트 간격 | 8px |
| 정렬 | 수직/수평 중앙 |
