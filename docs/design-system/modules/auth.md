# Auth Design System

Atena Design System 기반의 인증 및 워크스페이스 진입 화면 컴포넌트 규격.
로그인/회원가입 후 워크스페이스를 선택하거나 새로 생성하여 앱에 진입한다.

> 기반: [`../DESIGN_SYSTEM.md`](../DESIGN_SYSTEM.md) / [`../tokens.json`](../tokens.json)
> 프리뷰: [`auth-preview.html`](./auth-preview.html)

## 1. Login Screen (로그인 화면)

앱 최초 진입 시 표시되는 로그인 화면. 이메일/비밀번호로 인증한다.

```
┌──────────────────────────────────────┐
│                                      │
│            [Logo 48px]               │
│              Atena                   │
│                                      │
│  ┌──────────────────────────────┐    │
│  │  이메일                       │    │
│  └──────────────────────────────┘    │
│  ┌──────────────────────────────┐    │
│  │  비밀번호                 [👁] │    │
│  └──────────────────────────────┘    │
│                                      │
│  ┌──────────────────────────────┐    │
│  │           로그인              │    │
│  └──────────────────────────────┘    │
│                                      │
│       계정이 없으신가요? 회원가입      │
└──────────────────────────────────────┘
```

### 1-1. 레이아웃

| 속성 | 값 |
|------|-----|
| 배경 | `background` |
| 정렬 | 수직 중앙, 수평 중앙 |
| 좌우 패딩 | 24px |

### 1-2. 로고 영역

| 속성 | 값 |
|------|-----|
| 아이콘 | `shield` 48px, `primary.600` |
| 앱 이름 | 24px, bold (700), `textPrimary` |
| 아이콘-이름 간격 | 12px |
| 하단 간격 | 48px (입력 필드와의 간격) |
| 정렬 | 수평 중앙 |

### 1-3. 입력 필드

| 속성 | 값 |
|------|-----|
| 높이 | 48px |
| 배경 | `surface` |
| 보더 | `border` 1px |
| Border Radius | 12px |
| 좌우 패딩 | 16px |
| 텍스트 | 15px, regular (400), `textPrimary` |
| 플레이스홀더 | 15px, regular (400), `textTertiary` |
| 필드 간 간격 | 12px |

**포커스 상태:**

| 속성 | 값 |
|------|-----|
| 보더 | `primary.600` 1px |

**비밀번호 필드:**

| 요소 | 규격 |
|------|------|
| 우측 토글 버튼 | `eye` / `eye-off` 20px, `textTertiary` |
| 터치 영역 | 44×44px |

### 1-4. 로그인 버튼

| 속성 | 값 |
|------|-----|
| 높이 | 48px |
| 배경 | `primary.600` |

| 정렬 | 중앙 |
| 상단 간격 | 24px (입력 필드와의 간격) |

**비활성 상태 (입력 미완료):**

|------|-----|

### 1-5. 회원가입 링크

│  └──────────────────────────────┘    │
│                                      │
│  ┌──────────────────────────────┐    │
│  │  +  새 워크스페이스 만들기     │    │
│  └──────────────────────────────┘    │

## 4. Workspace Create Screen (워크스페이스 생성 화면)

새 워크스페이스를 생성하는 화면. 최소한의 정보만 입력받는다.

```
┌──────────────────────────────────────┐
│  ←  워크스페이스 만들기                │
├──────────────────────────────────────┤
│                                      │
│         [🏢 64px 아이콘 영역]          │
│            로고 추가 (선택)            │
│                                      │
│  회사/팀 이름                          │
│  ┌──────────────────────────────┐    │
│  │  워크스페이스 이름              │    │
│  └──────────────────────────────┘    │
│                                      │
│                                      │
│  ┌──────────────────────────────┐    │
│  │          만들기                │    │
│  └──────────────────────────────┘    │
└──────────────────────────────────────┘
```

### 4-1. 헤더

2-1 회원가입 헤더와 동일한 규격. 제목만 "워크스페이스 만들기"로 변경.

### 4-2. 로고 설정 영역

| 속성 | 값 |
|------|-----|
| 상단 간격 | 32px |
| 정렬 | 수평 중앙 |

| 요소 | 규격 |
|------|------|
| 아이콘 영역 | 64×64px, `surface` 배경, 20px radius |
| 아이콘 | `building-2` 28px, `textTertiary` |
| 보더 | `border` 1px dashed |
| 안내 텍스트 | 13px, regular (400), `textTertiary` |
| 아이콘-텍스트 간격 | 8px |
| 탭 동작 | 이미지 선택 (선택 사항) |

### 4-3. 이름 입력

| 속성 | 값 |
|------|-----|
| 상단 간격 | 32px |
| 좌우 패딩 | 24px |

**라벨:**

| 속성 | 값 |
|------|-----|
| 텍스트 | 14px, semibold (600), `textPrimary` |
| 하단 간격 | 8px |

**입력 필드:** 1-3 입력 필드와 동일한 규격. 플레이스홀더 "워크스페이스 이름".

### 4-4. 만들기 버튼

1-4 로그인 버튼과 동일한 규격.

| 속성 | 값 |
|------|-----|
| 상단 간격 | 32px |
| 좌우 마진 | 24px |

라벨: "만들기"

# Part 2. 조합 가이드

---

## 전체 흐름

```
[1. Login Screen]
       │
       ├─ 로그인 성공 ──→ [3. Workspace Select Screen]
       │                        │
       │                        ├─ 워크스페이스 선택 ──→ 홈 화면 진입
       │                        │
       │                        └─ 새로 만들기 ──→ [4. Workspace Create Screen]
       │                                                    │
       │                                                    └─ 생성 완료 ──→ 홈 화면 진입
       │
       └─ 회원가입 ──→ [2. Register Screen]
                              │
                              └─ 가입 완료 ──→ [3. Workspace Select Screen]
```

### 자동 로그인

| 조건 | 동작 |
|------|------|
| 토큰 유효 + 워크스페이스 1개 | 홈 화면 직행 |
| 토큰 유효 + 워크스페이스 복수 | 워크스페이스 선택 화면 |
| 토큰 없음/만료 | 로그인 화면 |
