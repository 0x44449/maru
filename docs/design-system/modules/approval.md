# Approval Design System

Atena Design System 기반의 결재 화면 컴포넌트 규격.
그룹웨어의 전자결재 목록과 상세 화면을 제공한다.

> 기반: [`../DESIGN_SYSTEM.md`](../DESIGN_SYSTEM.md) / [`../tokens.json`](../tokens.json)
> 프리뷰: [`approval-preview.html`](./approval-preview.html)
>
> **상위 규격:** 아바타는 `DESIGN_SYSTEM.md` 규격을 따른다.
> **참조:** 홈 화면의 Approval Preview (home.md #5)에서 "더보기" 탭 시 이 화면으로 진입.

---

# Part 1. 컴포넌트

---

## 1. Approval Header (결재 헤더)

결재 화면 상단의 헤더. 화면 제목과 검색 버튼을 포함한다.

```
┌──────────────────────────────────────┐
│  결재                          [🔍]  │
│  20px                           44  │
└──────────────────────────────────────┘
             높이: 56px
```

### 1-1. 기본 규격

| 속성 | 값 |
|------|-----|
| 높이 | 56px |
| 배경 | `background` |
| 하단 보더 | `border` 1px |
| 좌측 패딩 | 16px |
| 우측 패딩 | 4px |
| 정렬 | 세로 중앙, 좌우 space-between |
| position | sticky, top: 0 |
| z-index | `sticky` (1100) |

### 1-2. 내부 요소

| 요소 | 규격 |
|------|------|
| 제목 | 20px, bold (700), `textPrimary` |
| 검색 버튼 | 44×44px 터치 영역, `search` 아이콘 24px, `textSecondary` |

---

## 2. Mailbox Selector (결재함 셀렉터)

현재 보고 있는 결재함을 표시하고, 탭 시 드롭다운으로 함을 전환한다.

```
┌──────────────────────────────────────┐
│  📥 결재 대기  ∨                 3건  │
└──────────────────────────────────────┘

        드롭다운 열림 시:
        ┌──────────────────────┐
        │  📥 결재 대기      ✓  │
        │  ✅ 처리 완료         │
        │  📝 기안             │
        │  👁 참조             │
        └──────────────────────┘
```

### 2-1. 셀렉터 바

| 속성 | 값 |
|------|-----|
| 높이 | 48px |
| 좌우 패딩 | 16px |
| 배경 | `background` |
| 하단 보더 | `border` 1px |
| 정렬 | 세로 중앙, 좌우 space-between |
| cursor | pointer |
| 탭 피드백 | `surface` 배경 |

### 2-2. 셀렉터 바 내부 요소

**좌측 영역:**

| 요소 | 규격 |
|------|------|
| 아이콘 | 20px, `textSecondary` |
| 아이콘-텍스트 간격 | 8px |
| 함 이름 | 15px, semibold (600), `textPrimary` |
| 텍스트-chevron 간격 | 4px |
| chevron | `chevron-down` 16px, `textTertiary` |

**우측 영역:**

| 요소 | 규격 |
|------|------|
| 건수 | 14px, regular (400), `textTertiary` |

**건수 표시:** `N건` 형식 (예: "3건"). 0건일 때는 미표시.

### 2-3. 드롭다운 메뉴

| 속성 | 값 |
|------|-----|
| 위치 | 셀렉터 바 하단, 좌우 16px 마진 |
| 배경 | `surfaceRaised` |
| 보더 | `border` 1px |
| Border Radius | 12px |
| 그림자 | `shadow.md` |
| 패딩 | 4px 0 |
| z-index | `popover` (1400) |

### 2-4. 드롭다운 메뉴 아이템

| 속성 | 값 |
|------|-----|
| 높이 | 48px |
| 좌우 패딩 | 16px |
| 아이콘 | 20px, `textSecondary` |
| 아이콘-텍스트 간격 | 12px |
| 텍스트 | 15px, regular (400), `textPrimary` |
| 탭 피드백 | `surface` 배경 |

**선택된 아이템:** 우측에 `check` 아이콘 16px, `primary.600`.

### 2-5. 결재함 목록

| 아이콘 | 함 이름 | 내용 | 기본 선택 |
|--------|---------|------|----------|
| `inbox` | 결재 대기 | 내가 결재해야 할 미처리 건 | ✅ |
| `check-circle` | 처리 완료 | 내가 승인/반려 처리한 건 | |
| `pen-line` | 기안 | 내가 올린 결재 (모든 상태) | |
| `eye` | 참조 | CC로 받은 결재 | |

### 2-6. 백드롭

| 속성 | 값 |
|------|-----|
| 배경 | 투명 |
| 동작 | 드롭다운 외부 탭 시 메뉴 닫기 |
| z-index | `popover` - 1 |

## 3. Approval Item (결재 아이템)

결재 목록의 개별 아이템.

```
┌──────────────────────────────────────┐
│  📄  휴가 신청서              대기 중  │
│      김도현 · 개발팀                   │
│      2024년 3월 10일                  │
└──────────────────────────────────────┘
```

### 3-1. 기본 규격

| 속성 | 값 |
|------|-----|
| 좌우 패딩 | 16px |
| 상하 패딩 | 14px |
| 배경 | `background` |
| 탭 피드백 | `surface` 배경 |
| 하단 보더 | `border` 1px |

### 3-2. 내부 레이아웃

```
┌─────────────────────────────────────┐
│ [Icon]  [텍스트 영역]         [뱃지]  │
│  20px   flex: 1             flex-end │
└─────────────────────────────────────┘
```

| 요소 | 규격 |
|------|------|
| 아이콘 | `file-text` 20px, `textTertiary`, 상단 정렬 (align-self: flex-start, margin-top: 2px) |
| 아이콘-텍스트 간격 | 12px |

### 3-3. 텍스트 영역

| 요소 | 규격 |
|------|------|
| 제목 | 15px, regular (400), `textPrimary`, 1줄 말줄임 |
| 기안자 정보 | 13px, regular (400), `textTertiary` |
| 날짜 | 13px, regular (400), `textTertiary` |
| 제목-기안자 간격 | 2px |
| 기안자-날짜 간격 | 2px |

**기안자 표시 형식:** `이름 · 부서` (예: "김도현 · 개발팀")
**날짜 표시 형식:** `YYYY년 M월 D일` (예: "2024년 3월 10일")

### 3-4. 상태 뱃지

home.md 5-3과 동일한 규격.

| 속성 | 값 |
|------|-----|
| 텍스트 | 12px, medium (500) |
| 패딩 | 4px 8px |
| Border Radius | 6px |
| 위치 | 우측 상단 정렬 (align-self: flex-start) |

| 상태 | 배경 | 텍스트 |
|------|------|--------|
| 대기 중 | `warning.light` | `warning.default` |
| 승인됨 | `success.light` | `success.default` |
| 반려됨 | `error.light` | `error.default` |

## 4. Empty State (빈 상태)

결재 목록이 비어있을 때 표시.

```
      [📄 48px]
    결재 내역이 없습니다
  현재 조건에 맞는 결재가 없어요
```

| 속성 | 값 |
|------|-----|
| 아이콘 | `file-check` 48px, `textTertiary` |
| 제목 | 16px, semibold (600), `textSecondary` |
| 설명 | 14px, regular (400), `textTertiary` |
| 아이콘-제목 간격 | 12px |
| 제목-설명 간격 | 4px |
| 정렬 | 수직/수평 중앙 |
| 최소 높이 | 나머지 영역 전체 (flex: 1) |

## 5. Approval Detail Screen (결재 상세 화면)

```
┌──────────────────────────────────────┐
│  ←  결재 상세                         │
├──────────────────────────────────────┤
│                                      │
│  휴가 신청서                          │
│  📄 휴가 신청                        │
│  김도현 · 개발팀                      │
│  2024년 3월 10일                     │
│                                      │
├──────────────────────────────────────┤
│  결재선                              │
│  ┌─────┐ ┌─────┐ ┌─────┐           │
│  │ 김도현│ │ 박지영│ │ 윤혜진│          │
│  │ 기안  │ │대기 중│ │  -   │          │
│  └─────┘ └─────┘ └─────┘           │
├──────────────────────────────────────┤
│  첨부파일                            │
│  📎 휴가신청서.pdf                    │
├──────────────────────────────────────┤
│           [반려]    [승인]            │
└──────────────────────────────────────┘
```

### 5-1. 상세 헤더

| 속성 | 값 |

| 좌측 패딩 | 4px |
| 우측 패딩 | 16px |
| 정렬 | 세로 중앙 |

| 요소 | 규격 |
|------|------|
| 뒤로가기 버튼 | 44×44px 터치 영역, `arrow-left` 24px, `textPrimary` |
| 제목 | 17px, semibold (600), `textPrimary` |
| 뒤로가기-제목 간격 | 4px |

### 5-2. 문서 정보 카드

| 속성 | 값 |
|------|-----|
| 배경 | `background` |
| 패딩 | 20px 16px |
| 하단 보더 | `border` 1px |

| 요소 | 규격 |

**상태 뱃지:** 제목 우측에 3-4번과 동일한 뱃지 표시.

### 5-3. 결재선 (Approval Line)

결재 프로세스의 단계를 시각화.

| 속성 | 값 |
|------|-----|
| 배경 | `background` |
| 패딩 | 16px |
| 하단 보더 | `border` 1px |

**섹션 제목:**

| 속성 | 값 |
|------|-----|
| 텍스트 | 13px, semibold (600), `textTertiary` |
| 하단 간격 | 12px |

**결재자 아이템:**

```
┌──────────┐    ┌──────────┐    ┌──────────┐
│ [Avatar] │ ── │ [Avatar] │ ── │ [Avatar] │
│   이름   │    │   이름   │    │   이름   │
│   상태   │    │   상태   │    │   상태   │
└──────────┘    └──────────┘    └──────────┘
```

| 속성 | 값 |
|------|-----|
| 레이아웃 | flex, 가로 배치 |
| 아이템 간격 | 연결선으로 연결 |
| 아이템 너비 | 균등 분배 |

**개별 결재자:**

| 속성 | 값 |
|------|-----|
| 정렬 | 수평 중앙 |
| 아바타 | 36px (`avatar.md`), 12px radius |
| 이름 | 13px, medium (500), `textPrimary` |
| 상태 | 12px, regular (400) |
| 아바타-이름 간격 | 8px |
| 이름-상태 간격 | 2px |

**결재자 상태:**

| 상태 | 텍스트 | 색상 |
|------|--------|------|
| 기안 | "기안" | `textTertiary` |
| 승인 | "승인" | `success.default` |
| 대기 중 | "대기 중" | `warning.default` |
| 반려 | "반려" | `error.default` |
| 미도달 | "-" | `textTertiary` |

**연결선:**

| 속성 | 값 |
|------|-----|
| 두께 | 1px |
| 색상 | `border` |
| 위치 | 아바타 중앙 높이, 아이템 사이 |

### 5-4. 첨부파일

| 속성 | 값 |
|------|-----|
| 배경 | `background` |
| 패딩 | 16px |
| 하단 보더 | `border` 1px |

**섹션 제목:** 5-3 섹션 제목과 동일.

**파일 아이템:**

### 5-5. 액션 바

결재 처리 버튼. 대기 중 상태의 수신 결재에만 표시.

| 속성 | 값 |
|------|-----|

| 상단 보더 | `border` 1px |
| 패딩 | 12px 16px |
| 정렬 | 세로 중앙 |

**버튼:**

| 버튼 | 배경 | 텍스트 | 기타 |

| 속성 | 값 |
|------|-----|
