# Contacts Design System

Atena Design System 기반의 연락처 화면 컴포넌트 규격.
그룹웨어의 조직 구성원 연락처를 부서별로 표시한다.

> 기반: [`../DESIGN_SYSTEM.md`](../DESIGN_SYSTEM.md) / [`../tokens.json`](../tokens.json)
> 프리뷰: [`contacts-preview.html`](./contacts-preview.html)
>
> **상위 규격:** 아바타는 `DESIGN_SYSTEM.md` 규격을 따른다.

# Part 1. 컴포넌트

---

## 1. Contacts Header (연락처 헤더)

연락처 화면 상단의 헤더. 화면 제목과 검색 버튼을 포함한다.

```
┌──────────────────────────────────────┐
│  연락처                    [🔍] [⚙]  │
│  20px                       44   44  │
└──────────────────────────────────────┘
             높이: 56px
```

### 1-1. 기본 규격

| 속성 | 값 |
|------|-----|
| 높이 | 56px |
| 배경 | `background` |
| 하단 보더 | `border` 1px |
| 좌측 패딩 | 16px |
| 우측 패딩 | 4px |
| position | sticky, top: 0 |
| z-index | `sticky` (1100) |

### 1-2. 내부 요소

| 요소 | 규격 |
|------|------|
| 제목 | 20px, bold (700), `textPrimary` |
| 우측 버튼들 | 각 44×44px 터치 영역, 아이콘 24px |
| 아이콘 색상 | `textSecondary` |

**우측 버튼 구성:** `[검색 search]` `[설정 settings]`

- 제목은 **좌측 정렬**, 우측 버튼은 **우측 정렬**
- 제목 영역은 `flex: 1`로 남은 공간 차지

### 1-3. 설정 버튼 동작

설정 버튼 탭 시 Settings Popup Menu(5번) 표시.

## 2. Section Header (섹션 헤더)

"즐겨찾기", "개발팀", "디자인팀" 등 섹션을 구분하는 헤더. 접기/펼치기 기능 포함.

```
┌──────────────────────────────────────┐
│  개발팀 5                        [∨] │
└──────────────────────────────────────┘
```

### 2-1. 기본 규격

| 속성 | 값 |
|------|-----|
| 높이 | 40px |
| 좌우 패딩 | 16px |
| 배경 | `background` |
| 정렬 | 세로 중앙, 좌우 space-between |

### 2-2. 내부 요소

| 요소 | 규격 |
|------|------|
| 섹션명 | 13px, semibold (600), `textTertiary` |
| 카운트 | 13px, regular (400), `textTertiary`, 섹션명 우측 4px 간격 |
| 접기/펼치기 아이콘 | `chevron-down` 16px, `textTertiary`, 우측 정렬 |

### 2-3. 접기/펼치기

| 상태 | 아이콘 | 동작 |
|------|--------|------|
| 펼침 (기본) | `chevron-down` | 목록 표시 |
| 접힘 | `chevron-up` | 목록 숨김, 섹션 헤더만 표시 |

아이콘은 탭 시 180° 회전 애니메이션 (`transition.normal` 200ms).

## 3. Contact Item (연락처 아이템)

조직 구성원의 개별 아이템.

```
┌──────────────────────────────────────┐
│  [Avatar]  홍길동                     │
│   36px    백엔드 개발                 │
└──────────────────────────────────────┘
```

### 3-1. 기본 규격

| 속성 | 값 |
|------|-----|
| 높이 | 56px |
| 좌우 패딩 | 16px |
| 배경 | `background` |
| 탭 피드백 | `surface` 배경 |

### 3-2. 아바타

| 속성 | 값 |
|------|-----|
| 크기 | 36px (`avatar.md`) |
| Border Radius | 12px |
| 아바타-텍스트 간격 | 12px |

### 3-3. 텍스트

| 속성 | 값 |
|------|-----|
| 이름 | 15px, regular (400), `textPrimary`, 1줄 말줄임 |
| 직책/역할 | 13px, regular (400), `textTertiary`, 1줄 말줄임 |
| 이름-직책 간격 | 2px |
| 직책 미설정 시 | 직책 줄 미표시, 이름만 세로 중앙 |

## 4. Empty State (빈 상태)

연락처 목록이 비어있을 때 표시.

```
      [📒 48px]
   조직 구성원이 없습니다
```

| 속성 | 값 |
|------|-----|
| 아이콘 | `book-user` 48px, `textTertiary` |
| 제목 | 16px, semibold (600), `textSecondary` |
| 설명 | 14px, regular (400), `textTertiary` |
| 아이콘-제목 간격 | 12px |
| 제목-설명 간격 | 4px |
| 정렬 | 수직/수평 중앙 |

## 5. Settings Popup Menu (설정 팝업 메뉴)

헤더의 설정 버튼 탭 시 표시되는 드롭다운 팝업 메뉴.

```
                    ┌─────────────────┐
                    │ ✏️ 즐겨찾기 편집  │
                    │ ↕️ 정렬          │
                    └─────────────────┘
```

### 5-1. 컨테이너

| 속성 | 값 |
|------|-----|
| 위치 | 설정 버튼 하단 우측 정렬 |
| 최소 너비 | 180px |
| 배경 | `surfaceRaised` |
| 보더 | `border` 1px |
| Border Radius | 12px |
| 그림자 | `shadow.md` |
| 패딩 | 4px 0 |
| z-index | `popover` (1200) |

### 5-2. 메뉴 아이템

| 속성 | 값 |
|------|-----|
| 높이 | 44px |
| 좌우 패딩 | 16px |
| 아이콘 | 20px, `textSecondary` |
| 아이콘-텍스트 간격 | 12px |
| 텍스트 | 14px, regular (400), `textPrimary` |
| 탭 피드백 | `surface` 배경 |

### 5-3. 메뉴 항목

| 아이콘 | 라벨 | 동작 |
|--------|------|------|
| `pencil` | 즐겨찾기 편집 | 즐겨찾기 설정 편집 모드 진입 |
| `arrow-up-down` | 정렬 | 정렬 옵션 (이름순/부서순) |

### 5-4. 백드롭

| 속성 | 값 |
|------|-----|
| 배경 | 투명 |
| 동작 | 메뉴 외부 탭 시 메뉴 닫기 |
| z-index | `popover` - 1 |

## 전체 레이아웃

```
┌──────────────────────────────┐
│  [1. Contacts Header] ─ 고정  │
├──────────────────────────────┤
│  [2. Section: 즐겨찾기]       │
│  [3. Contact Item] ×N       │
│  [2. Section: 개발팀]        │
│  [3. Contact Item] ×N       │
│  [2. Section: 디자인팀]      │
│  [3. Contact Item] ×N       │
│  [2. Section: 관리팀]        │
│  [3. Contact Item] ×N       │
│         (스크롤 영역)         │
└──────────────────────────────┘
```

### 스크롤 영역

| 속성 | 값 |
|------|-----|
| 범위 | Header 아래 전체 |
| 배경 | `background` |
| overflow | `overflow-y: auto` |

### 즐겨찾기 섹션

| 조건 | 동작 |
|------|------|
| 즐겨찾기 있음 | 즐겨찾기 섹션 최상단 표시 후 부서 섹션 표시 |
| 즐겨찾기 없음 | 즐겨찾기 섹션 미표시, 부서 섹션만 표시 |
