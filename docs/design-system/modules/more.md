# More Design System

Atena Design System 기반의 더보기 화면 컴포넌트 규격.
하단 탭 바의 마지막 탭으로, 프로필 관리 진입과 계정/앱 정보, 로그아웃 기능을 제공한다.

> 기반: [`../DESIGN_SYSTEM.md`](../DESIGN_SYSTEM.md) / [`../tokens.json`](../tokens.json)
> 프리뷰: [`more-preview.html`](./more-preview.html)
>
> **상위 규격:** 아바타는 `DESIGN_SYSTEM.md` 규격을 따른다.

---

# Part 1. 컴포넌트

---

## 1. More Header (더보기 헤더)

더보기 화면 상단의 헤더. 화면 제목만 표시한다.

```
┌──────────────────────────────────────┐
│  더보기                               │
│  20px                                │
└──────────────────────────────────────┘
             높이: 56px
```

### 1-1. 기본 규격

| 속성 | 값 |
|------|-----|
| 높이 | 56px |
| 배경 | `background` |
| 하단 보더 | `border` 1px |
| 좌측 패딩 | 16px |
| 우측 패딩 | 16px |
| 정렬 | 세로 중앙 |
| position | sticky, top: 0 |
| z-index | `sticky` (1100) |

### 1-2. 내부 요소

| 요소 | 규격 |
|------|------|
| 제목 | 20px, bold (700), `textPrimary` |

---

## 2. Profile Section (프로필 섹션)

내 프로필 정보를 보여주는 영역. 탭 시 프로필 관리 화면으로 진입한다.

```
┌────────────────────────────────────┐
│  [Avatar]  김도현               >  │
│   48px    개발팀 · 백엔드 개발      │
└────────────────────────────────────┘
```

### 2-1. 컨테이너

| 속성 | 값 |
|------|-----|
| 배경 | `background` |
| 패딩 | 16px |
| 탭 피드백 | `surface` 배경 |
| 커서 | pointer |

### 2-2. 내부 요소

| 요소 | 규격 |
|------|------|
| 아바타 | 48px (`avatar.lg`), 16px radius |
| 이름 | 16px, semibold (600), `textPrimary` |
| 소속 · 직책 | 13px, regular (400), `textTertiary` |
| 아바타-텍스트 간격 | 12px |
| 이름-소속 간격 | 2px |
| 우측 아이콘 | `chevron-right` 16px, `textTertiary` |
| 정렬 | 세로 중앙, 좌우 space-between |

**소속 표시 형식:** `부서명 · 직책` (예: "개발팀 · 백엔드 개발")

### 2-3. 탭 동작

프로필 섹션 탭 시 프로필 관리 화면으로 진입.

> 참고: `home.md` 7번 My Profile Screen과 동일한 화면으로 진입할 수 있다.

## 3. Menu Group (메뉴 그룹)

관련 메뉴 항목들을 그룹으로 묶어 표시한다.

```
┌────────────────────────────────────┐
│  👤  계정 정보                   >  │
├ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─┤
│  ⚙️  설정                       >  │
├ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─┤
│  ℹ️  앱 정보            v1.0.0  >  │
└────────────────────────────────────┘
```

### 3-1. 그룹 컨테이너

| 속성 | 값 |
|------|-----|
| 배경 | `background` |
| 상단 마진 | 8px |
| 보더 상하 | `border` 1px |

### 3-2. 메뉴 아이템

| 속성 | 값 |
|------|-----|
| 높이 | 48px |
| 좌우 패딩 | 16px |
| 정렬 | 세로 중앙, 좌우 space-between |
| 탭 피드백 | `surface` 배경 |
| 커서 | pointer |

| 요소 | 규격 |
|------|------|
| 아이콘 | 20px, `textSecondary` |
| 아이콘-텍스트 간격 | 12px |
| 텍스트 | 15px, regular (400), `textPrimary` |
| 우측 아이콘 | `chevron-right` 16px, `textTertiary` |

### 3-3. 아이템 간 구분선

| 속성 | 값 |
|------|-----|
| 높이 | 1px |
| 배경 | `border` |
| 좌측 들여쓰기 | 48px (아이콘 영역 우측 기준) |

### 3-4. 메뉴 항목

| 아이콘 | 라벨 | 동작 |
|--------|------|------|
| `circle-user-round` | 계정 정보 | 계정 정보 화면 진입 |
| `settings` | 설정 | 설정 화면 진입 |
| `info` | 앱 정보 | 앱 정보 화면 진입 |

### 3-5. 앱 정보 아이템 우측 버전 텍스트

앱 정보 아이템에는 chevron 좌측에 현재 버전을 표시한다.

| 요소 | 규격 |
|------|------|
| 버전 텍스트 | 13px, regular (400), `textTertiary` |
| 버전-chevron 간격 | 4px |

**표시 형식:** `v버전` (예: "v1.0.0")

```

└────────────────────────────────────┘
```

### 4-1. 컨테이너

| 속성 | 값 |
|------|-----|
| 배경 | `background` |
| 상단 마진 | 8px |
| 보더 상하 | `border` 1px |

### 4-2. 버튼

| 속성 | 값 |
|------|-----|
| 높이 | 48px |
| 좌우 패딩 | 16px |
| 정렬 | 세로 중앙 |
| 탭 피드백 | `surface` 배경 |
| 커서 | pointer |

| 요소 | 규격 |
|------|------|
| 아이콘 | `log-out` 20px, `error.default` |
| 아이콘-텍스트 간격 | 12px |
| 텍스트 | 15px, regular (400), `error.default` |

### 4-3. 탭 동작

로그아웃 확인 다이얼로그 표시 후 로그아웃 처리.

# Part 2. 조합 가이드

---

## 전체 레이아웃

```
┌──────────────────────────────┐
│  [1. More Header] ─── 고정    │
├──────────────────────────────┤
│  [2. Profile Section]        │
│                              │
│  [3. Menu Group]             │
│                              │
│  [4. Logout Button]          │
│                              │
│         (스크롤 영역)         │
│                              │
│  하단 safe area 여백          │
└──────────────────────────────┘
```

### 스크롤 영역

| 속성 | 값 |
|------|-----|
| 배경 | `surface` |
| 범위 | Header 아래 전체 |
| overflow | `overflow-y: auto` |
| 하단 패딩 | 24px (safe area) |

### 배치 순서

1. **Profile Section** — 최상단, 헤더 바로 아래
2. **Menu Group** — 프로필 아래 8px 간격 (계정 정보 / 설정 / 앱 정보)
3. **Logout Button** — 메뉴 그룹 아래 8px 간격
