# Workspace Design System (구 Home)

Atena Design System 기반의 모바일 워크스페이스 탭 화면 컴포넌트 규격.
프로필 유형(개인/워크스페이스)에 따라 다른 뷰를 표시한다.

- **개인 프로필**: 내 프로필 정보 + 소속 워크스페이스 목록 (알림 표시)
- **워크스페이스 프로필**: 워크스페이스 프로필 정보 + 근무/결재/공지 등 워크스페이스 콘텐츠

> 기반: [`../DESIGN_SYSTEM.md`](../DESIGN_SYSTEM.md) / [`../tokens.json`](../tokens.json)
> 프리뷰: [`home-preview.html`](./home-preview.html)
>
> **상위 규격:** 아바타는 `DESIGN_SYSTEM.md` 규격을 따른다.

# Part 1. 컴포넌트

---

## 1. Workspace Header (워크스페이스 헤더)

화면 제목과 설정 버튼으로 구성된 워크스페이스 상단 헤더.

```
┌──────────────────────────────────────┐
│  워크스페이스                    [⚙]  │
│  20px                           44  │
└──────────────────────────────────────┘
             높이: 56px
```

### 1-1. 기본 규격

| 속성 | 값 |
|------|-----|
| 높이 | 56px |
| 배경 | `background` |
| 하단 보더 | `border` 1px |
| 좌측 패딩 | 16px |
| 우측 패딩 | 4px |
| 정렬 | 세로 중앙, 좌우 space-between |
| position | sticky, top: 0 |
| z-index | `sticky` (1100) |

### 1-2. 내부 요소

| 요소 | 규격 |
|------|------|
| 제목 | 20px, bold (700), `textPrimary` |
| 설정 버튼 | 44×44px 터치 영역, `settings` 아이콘 24px, `textSecondary` |

## 2. Profile Banner (프로필 배너)

내 프로필 정보와 소속 조직을 보여주는 배너 카드. 탭 시 내 프로필 화면(7번)으로 진입.

```
┌────────────────────────────────────┐
│  [Avatar]  김도현               >  │
│   48px    개발팀 · 백엔드 개발      │
│                                    │
│  🏢  주식회사 아테나               │
└────────────────────────────────────┘
```

### 2-1. 카드 규격

| 속성 | 값 |
|------|-----|
| 좌우 마진 | 16px |
| 상단 마진 | 8px |
| 배경 | `background` |
| Border Radius | 16px |
| 보더 | `border` 1px |
| 패딩 | 16px |

### 2-2. 프로필 영역

| 속성 | 값 |
|------|-----|
| 정렬 | 좌측, 세로 중앙 |
| 탭 피드백 | `surface` 배경 |

| 요소 | 규격 |
|------|------|
| 아바타 | 48px (`avatar.lg`), 16px radius |
| 이름 | 16px, semibold (600), `textPrimary` |
| 소속 · 직책 | 13px, regular (400), `textTertiary` |
| 아바타-텍스트 간격 | 12px |
| 이름-소속 간격 | 2px |
| 우측 아이콘 | `chevron-right` 16px, `textTertiary` |

**소속 표시 형식:** `부서명 · 직책` (예: "개발팀 · 백엔드 개발")

### 2-3. 조직 정보

프로필 영역 하단에 소속 조직명 표시.

| 속성 | 값 |
|------|-----|

| 속성 | 값 |

| 상태 텍스트 | 14px, medium (500) |
| 인디케이터-텍스트 간격 | 8px |
| 정렬 | 좌측 |

**상태별 색상:**

| 상단 간격 | 16px (상태 표시와의 간격) |
| 하단 간격 | 16px (하단 영역과의 간격) |

**퇴근/근무 전 상태:** "00:00:00" 표시, `textTertiary` 색상.

### 3-4. 하단 영역

출근 시간과 액션 버튼이 좌우로 배치.

| 속성 | 값 |
|------|-----|
| 정렬 | 좌우 space-between, 세로 중앙 |

**출근 시간:**

| 속성 | 값 |
|------|-----|
| 시간 | 14px, medium (500), `textPrimary` |
| 라벨 | 14px, regular (400), `textTertiary` |
| 시간-라벨 간격 | 4px |

**액션 버튼:**

| 속성 | 값 |
|------|-----|
| 높이 | 40px |
| 좌우 패딩 | 20px |
| Border Radius | 10px |
| 텍스트 | 14px, semibold (600) |

| 상태 | 배경 | 텍스트 |
|------|------|--------|
| 출근 전 → 출근 | `primary.600` | `#ffffff` |
| 근무 중 → 퇴근 | `error.default` | `#ffffff` |
| 퇴근 후 | 미표시 |

## 4. Quick Stats (퀵 스탯)

미처리 항목 수를 숫자로 요약. 탭 시 해당 화면으로 이동.

```
┌──────────┬──────────┬──────────┐
│  결재     │  공지톡   │  근무시간  │
│    3     │    2     │ 24/40h  │
└──────────┴──────────┴──────────┘
```

### 4-1. 컨테이너

| 속성 | 값 |
|------|-----|
| 좌우 마진 | 16px |
| 상단 마진 | 8px |
| 레이아웃 | flex, 균등 분배 |
| 간격 | 8px |

### 4-2. 스탯 아이템

| 속성 | 값 |
|------|-----|
| 배경 | `background` |
| 보더 | `border` 1px |
| Border Radius | 12px |
| 패딩 | 14px 0 |
| 정렬 | 중앙 |
| 탭 피드백 | `surface` 배경 |

| 요소 | 규격 |
|------|------|
| 라벨 | 13px, regular (400), `textTertiary` |
| 숫자 | 20px, bold (700), `textPrimary` |
| 라벨-숫자 간격 | 4px |

**숫자가 0일 때:** `textTertiary` 색상.
**미처리 있을 때:** `textPrimary` 색상, 결재는 `primary.600`.

**근무시간 표시 형식:** `현재/목표h` (예: "24/40h"). 목표 초과 시 `error.default` 색상.

## 전체 레이아웃

프로필 유형에 따라 다른 레이아웃을 표시한다.

### 개인 프로필일 때

```
┌──────────────────────────────┐
│  [1. Workspace Header] ── 고정 │
├──────────────────────────────┤
│  [2. Profile Banner]         │
│  [내 워크스페이스 목록 카드]     │
│         (스크롤 영역)         │
│                              │
│  하단 safe area 여백          │
└──────────────────────────────┘
```

내 프로필 정보 + 소속 워크스페이스 목록을 표시. 각 워크스페이스에 미처리 알림(결재, 공지 등) 요약을 함께 보여준다.

### 워크스페이스 프로필일 때

```
┌──────────────────────────────┐
│  [1. Workspace Header] ── 고정 │
├──────────────────────────────┤
│  [2. Profile Banner]         │
│  [3. Work Clock]             │
│  [4. Quick Stats]            │
│  [5. Approval Preview]       │
│  [6. Notice Preview]         │
│         (스크롤 영역)         │
│                              │
│  하단 safe area 여백          │
```
