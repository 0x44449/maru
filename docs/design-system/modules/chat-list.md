# Chat List Design System

Atena Design System 기반의 채팅 목록 컴포넌트 규격.
디자인 시스템(`DESIGN_SYSTEM.md`)의 Notification Badge, Group Avatar를 사용한다.

> 기반: [`../DESIGN_SYSTEM.md`](../DESIGN_SYSTEM.md) / [`../tokens.json`](../tokens.json)
> 프리뷰: [`chat-list-preview.html`](./chat-list-preview.html)
>
> **상위 규격:** 아바타, Group Avatar, Notification Badge는 `DESIGN_SYSTEM.md` 규격을 따른다.

---

# Part 1. 컴포넌트

---

## 1. Chat List Item (채팅 목록 아이템)

채팅 목록의 핵심 컴포넌트. 각 대화방을 한 줄로 표현한다.

```
┌──────────────────────────────────────────────┐
│ [Avatar]  채팅방 이름        (뱃지)    오후 2:30 │
│  48px     마지막 메시지 미리보기...              │
└──────────────────────────────────────────────┘
```

### 1-1. 기본 규격

| 속성 | 값 |
|------|-----|
| 높이 | 72px |
| 좌우 패딩 | 16px |
| 상하 패딩 | 12px |
| 배경 | `background` |
| 탭 피드백 | `surface` 배경 (active/pressed 상태) |

### 1-2. 아바타 영역

| 속성 | 값 |
|------|-----|
| 크기 | 48px (`avatar.lg`) |
| Border Radius | 16px (둥근 사각형) |
| 아바타-텍스트 간격 | 12px |

### 1-3. 텍스트 영역

2줄 구조: 상단(이름 + 참여자 수), 하단(메시지 미리보기). 시간과 뱃지는 우측 영역(1-4)에 배치.

**상단 줄:**

| 속성 | 값 |
|------|-----|
| 채팅방 이름 | 16px, semibold (600), `textPrimary`, 1줄 말줄임 |
| 참여자 수 (그룹) | 13px, regular (400), `textTertiary`, 이름 우측 4px 간격 |
| 시간 | 13px, regular (400), `textTertiary`, 우측 정렬 |
| 이름-시간 간격 | 8px (최소) |

**하단 줄:**

| 속성 | 값 |
|------|-----|
| 메시지 미리보기 | 14px, regular (400), `textSecondary`, 1줄 말줄임 |
| 상단-하단 줄 간격 | 4px |

**시간 표시 형식:**

| 조건 | 표시 |
|------|------|
| 오늘 | "오후 2:30" (시간) |
| 어제 | "어제" |
| 올해 내 | "3월 15일" |
| 작년 이전 | "2023.12.25" |

### 1-4. 우측 영역

시간과 뱃지가 세로로 배치되는 영역.

| 속성 | 값 |
|------|-----|
| 정렬 | 우측, 상단 정렬 |
| 시간-뱃지 간격 | 6px |
| 뱃지 | `DESIGN_SYSTEM.md` Notification Badge |
| 최소 너비 | 수축 불가 (`flex-shrink: 0`) |

### 1-5. 상태 아이콘

채팅방 이름 좌측에 표시되는 상태 아이콘.

| 아이콘 | 설명 | 규격 |
|--------|------|------|
| 📌 고정됨 | `pin` 아이콘, 이름 좌측 | 14px, `textTertiary` |
| 🔇 음소거 | `bell-off` 아이콘, 이름 좌측 | 14px, `textTertiary` |
| 아이콘-이름 간격 | — | 4px |

### 1-6. 배리언트

| 배리언트 | 설명 |
|----------|------|
| 1:1 채팅 | 단일 아바타, 상대방 이름 |
| 그룹 채팅 | 그룹 아바타 (2번), 채팅방 이름 + 참여자 수 |
| 읽지 않음 | Notification Badge 표시, 채팅방 이름 bold 유지 |
| 모두 읽음 | 뱃지 미표시 |
| 고정됨 | 고정 아이콘 + `surface` 배경 |
| 음소거 | 음소거 아이콘, 뱃지 배경 `neutral.400` |

## 3. Chat List Filter (필터 탭)

채팅 목록 상단의 필터 칩. 가로 스크롤 가능.

```
[전체] [읽지 않음] [그룹] [1:1]
```

| 속성 | 값 |
|------|-----|
| 높이 | 32px |
| 좌우 패딩 | 12px |
| Font Size | 13px |
| Font Weight | medium (500) |
| Border Radius | `full` |
| 칩 간 간격 | 8px |
| 영역 좌우 패딩 | 16px |
| 영역 상하 패딩 | 8px |

### 상태

| 상태 | 배경 | 텍스트 | 보더 |
|------|------|--------|------|
| 비활성 | `surface` | `textSecondary` | `border` 1px |
| 활성 | `primary.600` | `#ffffff` | 없음 |

## 4. Swipe Actions (스와이프 액션)

목록 아이템을 좌로 스와이프 시 드러나는 액션 버튼들.

```
                        ┌────────┬────────┬────────┐
← 스와이프              │  고정   │ 음소거  │  나가기 │
                        └────────┴────────┴────────┘
```

### 4-1. 기본 규격

| 속성 | 값 |
|------|-----|
| 버튼 너비 | 72px |
| 버튼 높이 | 아이템 높이와 동일 (72px) |
| 아이콘 크기 | 20px |
| 라벨 | 11px, medium (500) |
| 아이콘-라벨 간격 | 4px |
| 정렬 | 세로 중앙 |

### 4-2. 액션 버튼

| 액션 | 아이콘 | 배경 | 텍스트/아이콘 색상 |
|------|--------|------|-------------------|
| 고정/해제 | `pin` / `pin-off` | `warning.default` | `#ffffff` |
| 음소거/해제 | `bell-off` / `bell` | `neutral.500` | `#ffffff` |
| 나가기 | `log-out` | `error.default` | `#ffffff` |

### 4-3. 동작

| 동작 | 설명 |
|------|------|
| 스와이프 임계값 | 72px (버튼 1개 너비) 이상 드래그 시 액션 노출 |
| 전체 스와이프 | 전체 너비 스와이프 시 첫 번째 액션(고정) 자동 실행 |
| 스냅 | 손 떼면 열린 상태 유지 또는 닫힘 |
| 다른 아이템 스와이프 | 기존 열린 아이템 자동 닫힘 |

## 5. Empty State (빈 상태)

채팅 목록이 비어있을 때 표시.

```
      [💬 48px]
   채팅을 시작해보세요
  새로운 대화를 시작해보세요
```

| 속성 | 값 |
|------|-----|
| 아이콘 | `message-circle` 48px, `textTertiary` |
| 제목 | 16px, semibold (600), `textSecondary` |
| 설명 | 14px, regular (400), `textTertiary` |
| 아이콘-제목 간격 | 12px |
| 제목-설명 간격 | 4px |
| 정렬 | 수직/수평 중앙 |

## 6. Chat List Header (채팅 목록 헤더)

채팅 목록 상단의 헤더. 화면 제목과 액션 버튼을 포함한다.

```
┌──────────────────────────────────────┐
│  채팅                [🔍] [＋] [⚙]  │
│  16px                 44   44   44   │
└──────────────────────────────────────┘
             높이: 56px
```

### 6-1. 기본 규격

| 속성 | 값 |
|------|-----|
| 높이 | 56px |
| 배경 | `background` |
| 하단 보더 | `border` 1px |
| 좌측 패딩 | 16px |
| 우측 패딩 | 4px |
| position | sticky, top: 0 |
| z-index | `sticky` (1100) |

### 6-2. 내부 요소

| 요소 | 규격 |
|------|------|
| 제목 | 20px, bold (700), `textPrimary` |
| 우측 버튼들 | 각 44×44px 터치 영역, 아이콘 24px |
| 아이콘 색상 | `textSecondary` |

**우측 버튼 구성:** `[검색 search]` `[채팅 추가 plus]` `[설정 settings]`

- 제목은 **좌측 정렬**, 우측 버튼은 **우측 정렬**
- 제목 영역은 `flex: 1`로 남은 공간 차지

### 6-3. 검색 버튼 동작

검색 버튼 탭 시 별도 통합 검색 화면으로 이동한다. 헤더 자체는 변하지 않음.

---

## 7. Settings Popup Menu (설정 팝업 메뉴)

헤더의 설정 버튼 탭 시 표시되는 드롭다운 팝업 메뉴.

```
                    ┌─────────────────┐
                    │ 📊 채팅방 정렬    │
                    │ ✏️ 채팅방 관리    │
                    │ ✓  모두 읽음 처리  │
                    │ 🔔 알림 설정      │
                    └─────────────────┘
```

### 7-1. 컨테이너

| 속성 | 값 |
|------|-----|
| 위치 | 설정 버튼 하단 우측 정렬 |
| 최소 너비 | 180px |
| 배경 | `surfaceRaised` |
| 보더 | `border` 1px |
| Border Radius | 12px |
| 그림자 | `shadow.md` |
| 패딩 | 4px 0 |
| z-index | `popover` (1200) |

### 7-2. 메뉴 아이템

| 속성 | 값 |
|------|-----|
| 높이 | 44px |
| 좌우 패딩 | 16px |
| 아이콘 | 20px, `textSecondary` |
| 아이콘-텍스트 간격 | 12px |
| 텍스트 | 14px, regular (400), `textPrimary` |
| 탭 피드백 | `surface` 배경 |

### 7-3. 메뉴 항목

| 아이콘 | 라벨 | 동작 |
|--------|------|------|
| `arrow-up-down` | 채팅방 정렬 | 정렬 옵션 서브메뉴 또는 바텀시트 |
| `pencil` | 채팅방 관리 | 편집 모드 진입 (일괄 고정/음소거/삭제) |
| `check-check` | 모두 읽음 처리 | 모든 채팅방 읽음 표시, 확인 다이얼로그 |
| `bell` | 알림 설정 | 알림 설정 화면 이동 |

### 7-4. 구분선

메뉴 항목 사이에 선택적으로 구분선을 넣을 수 있다.

| 속성 | 값 |
|------|-----|
| 높이 | 1px |
| 배경 | `border` |
| 좌우 마진 | 16px |
| 상하 마진 | 4px |

### 7-5. 백드롭

| 속성 | 값 |
|------|-----|
| 배경 | 투명 |
| 동작 | 메뉴 외부 탭 시 메뉴 닫기 |
| z-index | `popover` - 1 |

# Part 2. 조합 가이드

---

## 목록 아이템 조합

```
[1. Chat List Item]
  └ 좌측: [`DESIGN_SYSTEM.md` Group Avatar] 또는 단일 아바타
  └ 중앙: 이름 줄 + 미리보기 줄
  └ 우측: 시간 + [`DESIGN_SYSTEM.md` Notification Badge]
  └ (스와이프): [4. Swipe Actions]
```

### 아이템 간 구분

| 속성 | 값 |
|------|-----|
| 구분선 | `border` 1px |
| 구분선 시작 | 아바타 우측 끝 기준 (좌측 76px 들여쓰기) |
| 마지막 아이템 | 구분선 없음 |

### 고정된 채팅 영역

| 속성 | 값 |
|------|-----|
| 배경 | `surface` |
| 고정/일반 구분선 | `border` 1px, 전체 너비 |
